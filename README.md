# AI Smart Defense Watch — Women Safety System

**Project:** AI-Powered Non-Lethal Smart Defense Watch for Women  
**Author:** Shikha Patel

## Overview
This project is a complete prototype of a wearable safety watch that uses AI to detect threats (sound + motion + heart-rate), automatically starts evidence recording, and sends SOS alerts with live location to emergency contacts and authorities.

## Features
- AI Threat Detection (scream/aggresion/motion anomalies)
- One-tap SOS + gesture SOS
- Live GPS tracking
- Automatic audio/video recording & cloud upload
- Non-lethal defense (vibration / micro-spray)
- Watch ↔ Phone Bluetooth sync
- Backend API to forward alerts to emergency contacts
